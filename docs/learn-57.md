#### 初始化
```
# 环境：要求node16以上
npm init vue@latest # 初始化

# 安装运行
npm install && npm run dev

# 构建
npm run build

```

#### Vue3 运行原理
```
Vue3 运行原理主要涉及以下几个方面：

Vue3 的组件化机制：Vue3 中的组件化机制是其运行的核心，每个组件都是一个独立的实体，具有自己的状态和生命周期。组件可以包含子组件，形成组件树结构。

Vue3 的响应式数据绑定：Vue3 通过使用 ES6 的 Proxy 对象实现响应式数据绑定。当组件中的数据发生变化时，会触发视图的更新。

Vue3 的虚拟 DOM：Vue3 中的虚拟 DOM 是一个 JavaScript 对象树，用于描述真实 DOM 树的结构和内容。当数据发生变化时，Vue3 会通过比较新旧虚拟 DOM 树的差异，最小化地修改真实 DOM 树，以提高性能。

Vue3 的编译器：Vue3 的编译器将模板转换为渲染函数，渲染函数会生成虚拟 DOM 树，进而更新视图。Vue3 的编译器支持模板的静态分析和编译时优化，提高了应用的性能。

Vue3 的生命周期钩子函数：Vue3 的生命周期钩子函数提供了多个时机让开发者可以介入组件的生命周期，从而进行一些自定义的操作。

综上所述，Vue3 运行的基本原理是，通过组件化机制、响应式数据绑定和虚拟 DOM 等技术，实现数据和视图的双向绑定，并提供编译器和生命周期钩子函数等工具，使开发者能够更方便地开发、调试和优化 Vue3 应用程序
```
#### Vue3目录解析
```
Vue3 初始化项目的含义是使用 Vue CLI 工具创建一个新的 Vue3 项目，包含了必要的项目结构和配置，以便开发者可以快速地搭建一个新的 Vue3 应用程序。初始化项目会创建以下目录和文件：

- node_modules: 存放项目依赖的 Node.js 模块。
- public: 存放静态资源，如 index.html、图片等。
- src: 存放源代码，包括 Vue3 组件、样式、脚本等。
- .gitignore: Git 版本控制忽略文件。
- babel.config.js: Babel 编译配置文件。
- package.json: 项目的配置文件，包含项目名称、版本号、依赖、脚本等信息。
- README.md: 项目的说明文件。
- package-lock.json: 存放项目依赖的具体版本信息。
```
